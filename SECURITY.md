# Security Policy

## Supported Versions

This project does not currently publish long-term supported releases.
We recommend using the latest version from the default branch.

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please report it privately.

- Contact: blingblingchen@126.com

Please include:
- A description of the issue and potential impact
- Steps to reproduce
- Affected versions/commits (if known)

We will acknowledge receipt and work on a fix as soon as possible.
